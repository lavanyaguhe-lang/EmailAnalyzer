Project Information

About PHISH-SHIELD

An email and URL threat analysis platform designed to detect phishing indicators and provide actionable risk intelligence.

Project Description

PHISH-SHIELD analyzes suspicious email text and links using a hybrid detection pipeline. It combines rule-based phishing indicators with ML-style scoring to classify threats as Secure, Suspicious, or Phishing and stores scan history in MySQL for audit and reporting.

Objectives

  • Detect phishing emails and URLs with explainable risk scoring.
  • Reduce false positives using calibrated rule and model thresholds.
  • Persist scan events in a database for traceability and forensic review.
  • Provide a simple UI for security analysts to monitor and investigate threats.

Technologies Used

Backend: Python, Flask
Frontend: HTML, CSS, Jinja2
Database: MySQL
NLP/ML: TextBlob + Hybrid Scoring
Security: Session cookies, hashed passwords
Reporting: Chart-based risk visualization
